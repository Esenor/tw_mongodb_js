# tw_mongo_db_js

Basic usage of MongoDB with node js

------
## Requirements

- Node
- Docker

------
## Installation

```bash
docker-compose -f etc/docker/docker-compose up -d
npm install
```

------
## Usage

```bash
npm start
```

------
## dependencies

### NPM

- mongodb: `^3.5.8`
  - [npmjs.com/package/mongodb](https://www.npmjs.com/package/mongodb)

### DockerHub

- mongo: `latest`
  - [hub.docker.com/_/mongo](https://hub.docker.com/_/mongo)
